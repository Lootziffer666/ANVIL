# Principles

Shared product, UX, and development principles used across ANVIL projects.

## Core Threads

- Frictionless design without manipulative conversion logic
- State surfaces instead of dashboards
- Visibility as a function of semantic load
- Clear anchors over dense interfaces
- CI-first gated agent development
- knownbugs as memory, not shame
- Shared vision first; shared code only after proof
