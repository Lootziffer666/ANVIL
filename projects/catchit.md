# CatchIt

CatchIt is the origin point for ANVIL's state-surface UX grammar.

## Role In ANVIL

CatchIt contributes the design doctrine:

- state surfaces instead of dashboards
- anchor-driven UI
- progressive disclosure
- anti-dashboard thinking
- motion as semantic transition
- user state and semantic load drive visibility

## Important Boundary

Other ANVIL projects may use CatchIt's principles without copying CatchIt's visual style.
