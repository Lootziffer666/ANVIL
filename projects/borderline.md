# Borderline

Borderline is a Zen-OS-style Android context and friction-reduction project.

## Role In ANVIL

Borderline explores:

- context modules instead of app-icon navigation
- friction reduction across existing Android apps
- intuitive overlay / shortcut surfaces
- optional invisibility as a design virtue

## Boundary

Borderline's platform and permission risks are higher than normal apps. Shared ANVIL principles can guide it, but implementation must remain heavily gated.
