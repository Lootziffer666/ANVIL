# Core Gated Template

Minimal Pflicht-Template für ANVIL-Gate-Projekte.

## Enthalten

- Root-Guides: `README.md`, `AGENTS.md`, `claude.md`, `instructions.md`, `knownbugs.md`
- Handoff-Basis unter `handoff/`
- PR-Template unter `.github/PULL_REQUEST_TEMPLATE.md`
- Minimaler CI-Workflow unter `.github/workflows/ci.yml`

## Ziel

So klein wie möglich starten; optionale Module nur bei Bedarf zuschalten.
