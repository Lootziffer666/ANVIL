# Handoff Templates

Every gate creates or updates one handoff file.

Use:

```text
handoff/GATE_X_HANDOFF_TEMPLATE.md
```

Copy it to:

```text
handoff/GATE_0_HANDOFF.md
handoff/GATE_1_HANDOFF.md
...
```

A handoff is not optional. It is the project memory between agents, humans, and future sessions.
