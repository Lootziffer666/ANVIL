# GitHub Friction Log

Recurring GitHub workflow pain points, especially on mobile.

## Known Patterns

- mobile browser upload limits and awkward folder creation
- missing branch/PR visibility on small screens
- unclear agent push state
- credential loss in cloud coding environments
- PR title drift after continued branch work

## Rule

When GitHub friction blocks a workflow, capture:

- project
- device/context
- exact blocker
- workaround
- whether Painkiller / ANVIL should absorb this as a future helper pattern
