# Global knownbugs

Cross-project memory for recurring failures, pitfalls, and accepted risks.

This folder is not a replacement for project-local `knownbugs.md` files.

Project-local knownbugs track concrete bugs in one repo.

Global knownbugs track patterns that should harden future projects.

## Files

- `knownbugs.md` — global pattern log
- `github-friction.md` — recurring GitHub and mobile workflow friction
- `agent-failure-patterns.md` — recurring agent drift, auth, push, scope, and CI issues
