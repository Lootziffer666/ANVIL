# Template Extraction Sources

This folder captures reusable workflow, architecture, and documentation lessons extracted from OPENDORK, CATALON-GUARD, DEAFPIPER, and PAINKILLER.

It is not a migration of those projects.

It is a distillation layer for ANVIL templates.

## Source Repositories

- `Lootziffer666/OPENDORK`
- `Lootziffer666/CATALON-GUARD`
- `Lootziffer666/DEAFPIPER`
- `Lootziffer666/PAINKILLER`

## Output Areas

- `template-useful-patterns.md` — cross-project reusable patterns
- `template-exclusion-rules.md` — what should not be copied into templates
- `source-map.md` — which source contributed which pattern
